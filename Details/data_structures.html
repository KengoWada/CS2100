DATA STRUCTURES


In simple terms, data structures are structures programmed to store data, so that various operations can be carried out on it easily. It should be designed and implemented in a way that it reduces complexity and increases effeciency of a program.

Types of Data Structures

As we have discussed above, anything that can store data can be referred to as a data structure, therefore integers, strings, booleans and many others are all data structures. These are known as Primitive Data Structures.
